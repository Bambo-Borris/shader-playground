## Requirements
- A Compiler
- CMake (>=3.16)

## Build Instructions

```
cmake -B build -DBUILD_SHARED_LIBS="FALSE"
cmake --build build
```

## Run Instructions
### Windows
```
./build/Debug/shader-playground.exe
```

### MacOS & Linux 
```
./build/shader-playground
```
